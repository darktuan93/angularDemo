Animation trong Angular 1 cách đơn giản nhất

Animation Angular là thư viện có sẵn ở trong Angular, có thể viết animation angular theo kiểu inline bên trong component cũng được ví dụ như ảnh

Hoặc có thể viết thành 1 function và return về như hình

Và import vào component để sử dụng

ĐÂY LÀ ANIMATION THƯ VIỆN ANGULAR

Trong Animation Angular có rất nhiều thứ, có mấy cái cái cơ bản này cần nhớ

Từ Khóa Ý nghĩa Minh Họa
trigger Kích hoạt animation được khai báo [@animationTest2]="animationState" kích hoạt 1 thuộc tính trigger trong thẻ html bất kỳ
transition Dạng chuyển tiếp trạng thái :enter => :leave (chuyển từ trạng thái vào sang trạng thái thoát đi)
